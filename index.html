A little bit about me.

I am a front-end software engineer based in Ames, Iowa. My journey into tech started a bit rocky as I failed my first ever computer programming language course as a freshman in college! Since that class, I have always wanted to prove to myself that I could learn code. So I enrolled in a course online, and I was hooked! I built my first website two weeks after starting my journey.

That curiosity then opened a door that could never be shut. When I learned how to build my first website, I realized I found something that gave me the freedom & versatility I was looking for in my work. Now I've made a full switch to front-end development, where I can use my organizational skills and eye for detail to write clean, elegant code.

Hand-Coded HTML

My focus is writing clean, well-formatted, semantic HTML5 by hand to make sure the content is easy to read, easy to collaborate, trouble-shoot and accessible.

Well-Organized CSS

I pride myself on writing CSS that is easy to read and build on. I focus on keeping my CSS lean and fast to load, and I make it a habit to stay up to date on current best practices.
